# Available CLI Tools

The following advanced CLI tools are installed and available:

## File Search & Navigation
- fd - faster find alternative
- fzf - fuzzy finder
- tree - directory visualization

## Code Search & Analysis
- ripgrep (rg) - fast code search
- ast-grep - structural code search using AST patterns
- semgrep - semantic code analysis
- universal-ctags - symbol indexing

## Code Understanding
- tokei - code statistics
- git-delta - better diff visualization
- difftastic - structural/syntax-aware diffs

## Structured Data
- jq - JSON processing
- yq - YAML/TOML processing

## Automation & Benchmarking
- watchexec - run commands on file changes
- hyperfine - command benchmarking

## Network & APIs
- httpie - intuitive HTTP client
- websocat - WebSocket debugging

## Service tools
- gh - Github client

## Package management
- uv - use uv and uvx instead of pip
- pnpm - use pnpm instead of npm
